# Implementation Plan: Dimensions Input UX

**Branch**: `012-dims-input-ux` | **Date**: 2026-02-14 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/012-dims-input-ux/spec.md`

## Summary

Improve the lots table dimensions cell UX by adding floating header labels above each input ("Qty", "L", "W", "H", "Wgt"), styling separator tokens ("@", "x", "in", "lbs") with muted appearance (font-weight 400, slate-400 color), and widening inputs via `ch`-based `min-width` so values are never truncated. Pure template + CSS change — no backend or data model modifications.
